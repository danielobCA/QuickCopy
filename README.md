**Installation guide**
-----------------------

  Download latest release and unzip.
  
  For Google Chrome chrome://extensions/
  
  For Microsoft Edge edge://extensions/
  
  Turn on Developer mode.
  
  Click Load unpacked.
  
  Find and select the folder with the downloaded contents.
